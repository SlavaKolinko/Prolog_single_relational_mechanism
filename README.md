# Декартів добуток, θ-обмеження, θ-з’єднання та перетин у Prolog як прояви єдиного реляційного механізму

У роботі розглядається, як у Prolog декартів добуток, θ-обмеження, θ-з’єднання та перетин не є окремими операціями, а різними проявами одного механізму - інформаційної залежності або незалежності між викликами предикатів.

Показано, що те, що в алгебрі Кодда та SQL виглядає як різні оператори, у Prolog природно виникає через наявність або відсутність спільних змінних.

Мета роботи - продемонструвати цю ідею на прикладах Prolog та провести порівняння з алгеброю Кодда і SQL.
